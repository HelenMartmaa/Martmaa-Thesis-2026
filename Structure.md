# Struktuuriline ülevaade

## Pealkiri

**Veebirakenduse arendamine biomeditsiiniliste ***in vivo*** ja ***in vitro*** katsete planeerimiseks ning statistiliseks andmeanalüüsiks**

## Kirjaliku töö teooria osad

- Biomeditsiiniliste katsete planeerimine ja dokumenteerimine
- Statistilise analüüsi meetodid biomeditsiinis
- Veebirakenduste arhitektuur ja REST põhimõtted
- Statistiliste arvutuste teostamine Python keskkonnas
- Relatsioonilised andmebaasid ja andmete modelleerimine
- Ligipääsetavuse nõuded veebirakendustes

___

### Biomeditsiiniliste katsete planeerimine ja dokumenteerimine

- Ülevaade ***in vivo*** ja ***in vitro*** katsetest + planeerimisest
- Kontroll- ja eksperimentaalrühma ning valimi tähtsus + roll
- Andmete ja tulemuste dokumenteerimine
- Tulemuste reprodutseeritavus teaduses

### Statistilise analüüsi meetodid biomeditsiinis

- Kirjeldav statistika, graafikute kasutamine
- Normaalsuse hindamine
- Parameetriline vs mitteparameetriline analüüs
- p-väärtus, usaldusvahemikud
- t-test ja Mann-Whitney
- Survival analüüs

### Veebirakenduste arhitektuur ja REST põhimõtted

- Kliendi-serveri arhitektuur
- REST API
- Monoliitne vs mikroteenuste arhitektuur
- Frontend ja backend eraldatus
- JWT audentimine

### Statistiliste arvutuste teostamine Python keskkonnas

- Pythoni kasutamine teadusmaailmas, reprodutseeritavad arvutused
- NumPy, SciPy, StatsModels roll
- Teadusstandardid

### Relatsioonilised andmebaasid ja andmete modelleerimine

- Relatsiooniline andmebaas
- Normaliseerimine
- Primaar- ja võõrvõtmed
- Andmete terviklikkus

### Ligipääsetavuse nõuded veebirakendustes

- WCAG põhimõtted
- Semantiline HTML
- Klaviatuuriga navigeerimine
- Kontrastsus jne
- ARIA rollid
